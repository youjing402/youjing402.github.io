Instructions

Dear participant , please read the instructions before the experiment.

Experiment Overview

This experiment is about comparing the performance of traditional copy-paste technique (ctrl-c + ctrl-v) with a new copy-paste technique called AutoComPaste.

In this experiment, you would be given a piece of information (phrase, sentence or paragraph) in each trial. Your task would be to using the indicated technique to copy-paste the information.

Practice Trials

Before the actual main experiment, two practice trials will be shown to you to familiarize you with the usage of two techniques.

For traditional technique, search for the highlighted text and use ctrl-c + ctrl-v to copy and paste it to the text editor.

For AutoComPaste, after you have keyed in three or more characters, the autocomplete list will be shown to you. Upon choosing on the listing sentence, the sentence would be automatically appear in the text editor. You can then further edit on the text or append a new sentence as needed.

After the first two rounds, you automatically move to the actual experiment.

Main Experiment

You then continue with the main experiment, in which you need to work in the same way in practice trials. The main experiment contains 2 blocks and a total of 72 trials. It is shown as 76 trials because there include 2 practice trials and 2 break trials.
